# Stone Paper Scissor Game (Python)

# Description

This is a simple command-line game built using Python.
The user plays Stone Paper Scissor against the computer, which selects a random choice.

# Features

* User vs Computer gameplay
* Random computer choice
* Win/Lose/Draw result

# How to Run

Run the file using:

```bash
python main.py
```
